Projects

Split.It

Web Application

This project was developed during Fall 2018 for the Human-Computer Interaction course. For our project, me and another 2 group members created a high-fidelity prototype of a web-based system using web technologies such as HTML, CSS, JavaScript, and jQuery. Our website aims to help groups of friends, family, or co-workers to split a bill with ease. The user would simply go on the website and fill their information as a group and then decide which method they would like to split the bill.

The application could be used for anything related to paying in groups, such as restaurants, grocery stores, or splitting rent. If the group decided that one single person will pay for everybody, the system will randomly select a member.

The Computer Science Experience

Android Application

Thes project was developed during Summer 2019 for the Software Engeneering 1 course. Me and another 4 group members developed a text-based game for Android that simulates the life of a Computer Science student using Java and Android Studio. As the appointed UI developer of the application, I created the whole interface using XML, integrating it with backend. Spanning only 30 in-game days, The C.S. Experience is a short, simulation-RPG hybrid videogame in which players assume the role of a computer science student and are fast-tracked through a single university term that demands them to manage their time, character health, studies and other factors in order to pass their exams.

The weeks leading up to each exam period will present the player with a large degree of freedom and should be used to prepare for exams by engaging in character-developing activities. When it is time to write an exam, the player enters a turn-based style combat scenario, similar to Pokemon or Final Fantasy. A well-prepared player, who’s improved his or her skills by studying the course material, and has maintained his or her character’s happiness, sleep, and hunger values will have an easier ‘combat encounter’ with the exam, most likely receiving a higher grade.

Tripmate

React / React Native Application

Developed during the term of Winter 2020, TripMate aims to be the one-stop-shop for everything needed to plan the perfect trip. This web and mobile cross-platform application was developed alongside 8 group members using modern technologies such as React, React Native, MongoDB, Express.js and Node.js. During this project, every member applied feature ownership during the development of new features.

My responsibility for this project was the implementation of the flights feature. By working with both front and back-ends, my implementation integrates an external API which can retreive information about flights, such as status, delays, schedules, distance and flight-times.

TripMate’s organizational features include an itinerary, logbook, to-do list, budget tracker, and notification system. These features stand to help all travellers organization their plans in one location, which, ideally, will alleviate some of the stress of travelling.
